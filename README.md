# UsStreet

Parses and normalizes US streets. This does not parse full addresses, just the street.

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'us_street'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install us_street
